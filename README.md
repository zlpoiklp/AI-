\# AI 协同工作台



一个功能强大的多模型 AI 对话平台，支持同时与三个不同的 AI 模型进行对话，提供丰富的自定义功能和跨模型分析能力。



\## 🌟 核心功能



\### 1. \*\*三栏并行对话\*\*

\- 同时与三个不同的 AI 模型进行对话

\- 每个模型独立配置，支持不同服务商

\- 实时显示各模型状态和响应



\### 2. \*\*丰富的主题风格\*\*

\- 内置 9 种精美主题：

&nbsp; - 🌙 深邃（默认）

&nbsp; - 🧧 春节主题

&nbsp; - 🎄 圣诞主题

&nbsp; - 🎃 万圣节主题

&nbsp; - 💕 情人节主题

&nbsp; - 🥮 中秋主题

&nbsp; - 🌸 樱花主题

&nbsp; - 🌊 海洋主题

&nbsp; - ☀️ 明亮主题

\- 主题装饰动画效果

\- 支持节日氛围营造



\### 3. \*\*灵活的输入方式\*\*

\- \*\*统一输入区\*\*：一次性向选中的模型发送消息

\- \*\*独立输入框\*\*：每个模型栏下方都有独立的输入区域

\- \*\*快捷键支持\*\*：Enter发送，Shift+Enter换行

\- \*\*文件上传\*\*：支持图片和文本文件



\### 4. \*\*跨域分析功能\*\*

\- 引用其他模型的对话内容进行分析

\- 智能整合多个模型的回答

\- 可视化引用标记



\### 5. \*\*联网搜索模式\*\*

\- 为每个模型独立开启联网搜索

\- 全局联网搜索开关

\- 自动添加搜索提示词



\### 6. \*\*生图模式\*\*

\- 将文本转换为图像生成指令

\- 支持多种图像生成模型

\- 艺术风格描述优化



\### 7. \*\*自定义外观\*\*

\- \*\*消息气泡颜色\*\*：为每个模型自定义消息背景色

\- \*\*对话栏背景\*\*：支持纯色或自定义背景图

\- \*\*共用背景\*\*：上传一张图片作为三个栏的共同背景

\- \*\*字体大小\*\*：可调节 AI 回复字体大小（12-24px）



\### 8. \*\*快捷键系统\*\*

\- 可自定义的快捷键配置

\- 常用操作快捷键：

&nbsp; - `Ctrl+,`：打开/关闭设置

&nbsp; - `Ctrl+B`：切换共用背景

&nbsp; - `Ctrl+\\`：打开/关闭侧边栏

&nbsp; - `Ctrl+I`：显示/隐藏所有独立输入框

&nbsp; - `Ctrl+M`：最小化应用（桌面版）

\- 快捷键启用/禁用开关



\### 9. \*\*历史记录管理\*\*

\- 自动保存对话历史

\- 对话标题自动生成

\- 支持清空所有历史

\- 侧边栏快速切换对话



\## 🔧 API 配置指南



\### 支持的服务商



目前支持以下 AI 服务商：



1\. \*\*OpenAI\*\* - GPT系列模型

2\. \*\*Anthropic\*\* - Claude系列模型

3\. \*\*Google\*\* - Gemini系列模型

4\. \*\*DeepSeek\*\* - DeepSeek Chat/Coder模型

5\. \*\*通义千问\*\* - 阿里云大模型

6\. \*\*智谱AI\*\* - GLM系列模型

7\. \*\*硅基流动\*\* - 开源模型托管平台

8\. \*\*字节豆包\*\* - 字节跳动大模型

9\. \*\*OpenRouter\*\* - 多模型聚合平台

10\. \*\*自定义\*\* - 兼容 OpenAI API 的自定义端点



\### 配置步骤



\#### 1. 打开设置面板

点击右上角的设置按钮（齿轮图标）打开设置中心。



\#### 2. 配置 API 密钥和端点

为每个模型栏配置：



\- \*\*服务商选择\*\*：从下拉菜单中选择服务商

\- \*\*API Key\*\*：输入对应服务商的 API 密钥

\- \*\*Base URL\*\*：通常会自动填充，也可自定义

\- \*\*模型选择\*\*：从列表中选择或手动输入模型名称



\#### 3. 获取可用模型列表

点击"获取模型"按钮，平台会自动从 API 端点获取所有可用模型。



> \*\*注意\*\*：需要先填写正确的 API Key 和 Base URL 才能获取模型列表。



\#### 4. 测试连接

点击"测试"按钮验证配置是否正确，确保能正常连接到 API 服务。



\#### 5. 保存配置

配置完成后点击"保存配置"按钮，设置会自动保存到本地。

\## 🚀 快速开始



\### 1. 基本使用

1\. 在设置中配置至少一个模型的 API

2\. 在统一输入区输入问题

3\. 选择要发送的模型（默认为全选）

4\. 按 Enter 或点击发送按钮

5\. 查看三个模型的并行回答



\### 2. 跨域分析使用

1\. 点击某个模型输入区下方的"引用"按钮

2\. 勾选要引用的其他模型

3\. 输入问题并发送

4\. 该模型会基于其他模型的对话历史进行回答



\### 3. 文件分析

1\. 点击附件按钮上传图片或文本文件

2\. 多个模型可以同时分析同一批文件

3\. 支持预览和删除已上传文件



\### 4. 联网搜索

1\. 点击"联网"按钮开启搜索模式

2\. 发送问题，AI 会基于最新信息回答

3\. 可单独为每个模型开启，也可全局开启



\### 5. 生图模式

1\. 点击"生图"按钮切换模式

2\. 输入图像描述

3\. AI 会生成详细的图像生成提示词

\## 🔒 数据安全



\### 本地存储

\- 所有 API 密钥保存在浏览器本地存储中

\- 对话历史也保存在本地

\- 刷新页面或关闭浏览器不会丢失数据



\### 隐私保护

\- 不向第三方服务器发送对话内容（除了配置的 API 端点）

\- API 密钥仅用于与对应服务商通信

\- 可随时清除本地数据

\## ❓ 常见问题



\### Q: API 密钥安全吗？

A: API 密钥仅保存在您的浏览器本地存储中，不会发送到我们的服务器。请确保不与他人共享您的设备。



\### Q: 为什么获取模型列表失败？

A: 检查 API Key 和 Base URL 是否正确，确保网络连接正常，部分服务商可能有地域限制。



\### Q: 如何清除所有数据？

A: 在侧边栏底部点击"清空历史"，或清除浏览器本地存储。



\### Q: 支持中文吗？

A: 完全支持中文界面和中文对话，所有模型都支持多语言。



\### Q: 可以添加更多模型栏吗？

A: 目前固定为三个模型栏，未来版本可能考虑增加。



\### Q: 需要安装吗？

A: 不需要，这是一个纯网页应用，直接在浏览器中打开即可使用。
